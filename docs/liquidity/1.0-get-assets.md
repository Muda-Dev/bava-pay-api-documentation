# 1.0 Get Assets

Get available assets for a specific currency and service.

**Method:** GET  
**Endpoint:** `/accounts/getAssets`

**Query Parameters:**
- `q`: Query parameter (optional)

**Request Body:**
```json
{
    "amount": 400,
    "currency": "UGX",
    "asset_code": "USDT",
    "service_code": "MOBILE_MONEY"
}
```

**Response Example:**
```json
{
    "status": 200,
    "message": "success",
    "data": [
        {
            "asset_code": "USDT",
            "asset_name": "Tether USD",
            "available": true,
            "min_amount": 1,
            "max_amount": 1000000
        },
        {
            "asset_code": "USDC",
            "asset_name": "USD Coin",
            "available": true,
            "min_amount": 1,
            "max_amount": 1000000
        }
    ]
}
```
